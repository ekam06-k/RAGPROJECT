# RAG Pipeline using Google Gemini

## Features
- Load PDF
- Split into chunks
- Generate embeddings using Gemini
- Store embeddings in ChromaDB
- Retrieve relevant chunks
- Answer questions using Gemini

## Tech Stack
- Python
- LangChain
- ChromaDB
- Google Gemini API

## Run

pip install -r requirements.txt
python create_db.py
python query.py
